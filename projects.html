Past Projects

ec2stack

January 2014 - March 2014

ec2stack is an EC2 compatibility interface for Apache Cloudstack written in Python. This application allows utilities created for the Amazon EC2 API to be used against Apache Cloudstack. This effectively gives users their own private Amazon EC2-like Infrastructure.

gstack

August 2013 - September 2013

This project aims to create a new compute API set for Cloudstack that is compatible with Google Compute Engine. This would allow utilities created for the GCE API to be used against Apache Cloudstack.

Compendium Board Games

March 2013 - April 2013

The idea behind the project was to create a selection of well known board games in C++, it was carried out as part of a second year module that dealt with Object Oriented Design.
